# Select
A select component allows the users to select a value or multiple values from the given list of options. The
implementation features:

- Single Select
- Multi Select
- Flexible customization of the components
- Tagging / Creation of options
- Filtering with Fuzzy Search via [fuse.js](https://github.com/reaviz/react-use-fuzzy)
- Highlighting of selected options
- Async Support
- Groups Support
- Animation

## CSS Classes
The select component exposes the following CSS class names for custom
styling.

- `select-input`
- `select-input-value`
- `select-input-refresh`
- `select-input-clear`
- `select-input-toggle`
- `select-input-chip`
- `select-menu`
- `select-menu-item`
- `select-menu-create-option`
- `select-menu-empty-search`
- `select-menu-empty`
- `select-menu-group`
- `select-menu-group-header`
