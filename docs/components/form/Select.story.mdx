# Select
A select component that features:

- Single Select
- Multi Select
- Tag / Creation of options
- Filtering with Fuzzy Search
- Async Support
- Groups Support
- Animation

## Styling

### CSS Classes
The select component exposes the following CSS class names for custom
styling.

- `select-input`
- `select-input-value`
- `select-input-refresh`
- `select-input-clear`
- `select-input-toggle`
- `select-input-chip`
- `select-menu`
- `select-menu-item`
- `select-menu-create-option`
- `select-menu-empty-search`
- `select-menu-empty`
- `select-menu-group`
- `select-menu-group-header`

### CSS Variables
The select component exposes the following CSS variables for customization.

there already.

- `--select-input-background`
- `--select-input-border`

- `--select-input-disabled-color`
- `--select-input-icon-color`
- `--select-menu-background`
