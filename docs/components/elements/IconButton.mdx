This component uses the following design tokens (same as the [Button](?path=/docs/docs-components-elements-button--docs) component):

- `button-font-weight`
- `button-font-family`
- `button-background`
- `button-background-hover`
- `button-color`
- `button-color-hover`
- `button-color-on-background`
- `button-disabled-color-on-background`
- `button-border-radius`
- `button-border`
- `button-focus`
- `button-focus-offset`
- `button-spacing-sm`
- `button-spacing-md`
- `button-spacing-lg`
- `button-adornment-size-sm`
