# Data Size
Data size is a component that renders bytes in a human readable format using
[human-format](https://www.npmjs.com/package/human-format).

You can use the component directly like in the example below or via
the API that is exported.

## Example

### Component
<Canvas>
  <Story id="data-data-size--simple" />
</Canvas>

### Function API
```ts
import { formatSize, FormatSizeTypes } from 'reablocks';

function formatSomething(size: FormatSizeTypes) {
  // size = number | string | null | undefined;
  return formatSize(size);
}
```
