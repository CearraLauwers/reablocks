# Duration
Duration is a component that renders duration of epoch time in a human readable format using
[human-format](https://www.npmjs.com/package/human-format).

You can use the component directly like in the example below or via
the API that is exported.

## Example
<Canvas sourceState="shown" of={DurationStories.Simple} />

### Function API
```ts
import { formatDuration, DurationFormatTypes } from 'reablocks';

function formatSomething(size: DurationFormatTypes) {
  // size = number | string | null | undefined;
  return formatDuration(size);
}
```
