# Notification

## CSS Variables
The notification component uses the following CSS variables.
If you are using the `DesignTokenProvider` these will be defined
in the `component` section.

- `--notification-background`
- `--notification-color`
- `--notification-border`
- `--notification-color-error`
- `--notification-color-warning`
- `--notification-color-success`
