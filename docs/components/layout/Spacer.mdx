# Spacer
Component that sets component spacing. It can be used to add
space between components such as aa menu.

## Design Tokens
This component uses the following design tokens:

- `spacer-gap`
- `spacer-dense-gap`

Learn more about design tokens in the [design tokens documentation](?path=/story/docs-design-tokens-getting-started--page).
