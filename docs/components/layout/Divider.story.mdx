# Divider
A simple divider component that supports horizontal and vertical.

## Usage
This component uses the following design tokens:

- `divider-spacing`: Margins to be applied to the component.
- `divider-bg`: Color to be applied to the component.

If you are using the `DesignTokensProvider`, you can override these using
the following namespace:

```ts
import { DesignTokens } from 'reablocks';

const customTheme: DesignTokens = {
  ...yourTheme,
  components: {
    divider: {
      'divider-spacing': yourTheme.spacings.md,
      'divider-bg': yourTheme.colors.gray['500']
    }
  }
};
```
