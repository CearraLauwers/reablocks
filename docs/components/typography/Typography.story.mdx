# Typography
Component for styling text and headings. It contains a set
of components for different heading sizes and text styles.

- `PageTitle` - The page title component is used for the main title of a page.
- `PrimaryHeading` - The primary heading component is used for the main heading of a section.
- `SecondaryHeading` - The secondary heading component is used for the secondary heading of a section.
- `Sub` - The sub component is used for a sub heading of a section.
- `Text` - The text component is used for normal text.

## Design Tokens
The typography component uses the following design tokens:

- `page-title-font-size`
- `page-title-font-weight`
- `page-title-color`
- `page-title-margin`
- `primary-heading-font-size`
- `primary-heading-font-weight`
- `primary-heading-color`
- `primary-heading-margin`
- `secondary-heading-font-size`
- `secondary-heading`
- `secondary-heading-font-weight`
- `secondary-heading-color`
- `secondary-heading-margin`
- `small-heading-font-size`
- `small-heading-font-weight`
- `small-heading-color`
- `small-heading-margin`
- `sub-font-size`
- `sub-font-weight`
- `sub-color`
- `sub-margin`

Learn more about design tokens in the [design tokens documentation](?path=/story/docs-design-tokens-getting-started--page).
