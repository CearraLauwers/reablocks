# Getting Started

## Installing

You can install reablocks with [NPM](https://www.npmjs.com/package/reablocks) or Yarn.

- NPM: `npm install reablocks --save`
- YARN: `yarn add reablocks`

## Useage

Some of the components require design tokens. You can use our
design token provider or you can define them manually via CSS
variables. Each component will declare what CSS variables it uses
in its documentation.

If you wish to use the provider, an example implementation might look
like so:

```tsx
import { DesignTokensProvider, lightTheme } from 'reablocks';

export const App = () => (
  <>
    <DesignTokensProvider theme={lightTheme}>
      Your Application Code
    </DesignTokensProvider>
  </>
)
```
